# About YGUI

This plugin provide extensions for UGUI

# Installing YGUI

YGUI is a Package and is installed through the Packages window in Unity. Consult the [Packages window documentation](https://docs.unity3d.com/Manual/upm-ui.html) for more information.

# Technical details
## Requirements

This version of YGUI is compatible with the following versions of the Unity Editor:

* 2019.4 and later (recommended)

## Package contents

The following table indicates the folder structure of the YGUI package:

|Location|Description|
|---|---|
|`<Runtime>`|Root folder containing the source for the YGUI Runtime. This is the source for what is available in the Player. |
|`<Editor>`|Root folder containing the source for the YGUI Editor used to edit YGUI files inside the Unity Editor.|
